cpproject
=========

cpproject provides a directory structure for your C++ project.

    .
    ├── CMakeLists.txt
    ├── README.md
    ├── src
    │   ├── CMakeLists.txt
    │   ├── main.cpp
    │   └── subdirectory
    │       ├── lib.cpp
    │       └── lib.h
    └── test
        ├── CMakeLists.txt
        └── subdirectory
            └── lib_test.cpp
